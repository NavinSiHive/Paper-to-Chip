# gds/
Generated GDSII layout.

- `pll_chip.gds[.gz]` — the full mixed-signal chip (top cell `pll_chip`, 1.90×1.90 mm).
- `PLLTOP_D40.gds[.gz]` — analog PLL hard macro (PLL core + on-macro ÷40).
- `pll_dcc.gds[.gz]` — hardened digital control core.

Compressed (`.gz`) in the public kit for the GitHub 100 MB file limit — `gunzip` to open in KLayout/magic.
